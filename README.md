# 밤하늘 천문대 · Night Sky 3D

내 위치와 현재 시각에서 실제로 보이는 밤하늘을 보여주는 인터랙티브 3D 웹 앱입니다.
빌드 과정 없는 **단일 `index.html`** (Three.js는 CDN 로드).

## 주요 기능
- 🌌 관측지·시각 기준 실제 별 위치 계산(약 110개 명칭성 + 배경 미광성)
- 🖱️ 드래그로 시야 회전 · 휠로 확대/축소 · 별에 마우스 오버 시 정보 표시
- ✳️ 별자리 선, 정보 있는 별/없는 별 시각 구분
- 🌍 3D 지구본에서 관측 위치 클릭 선택 (도시 프리셋·좌표 입력·내 위치도 지원)
- 🌙 실제 달 위치와 위상(초승달~보름~그믐), ☀️ 태양 고도로 낮/밤 표현
- ✨ 별 반짝임 · ☄️ 유성 연출
- ⏱️ 24시간 재생으로 별의 일주 궤적(트레일) 감상
- ⛰️ 지평선 지형과 시점 제한

## 로컬 실행
정적 파일이라 아무 정적 서버로 열면 됩니다. 위치 감지·텍스처 로드를 위해 `http(s)`로 서빙하세요.

```bash
python -m http.server 8137
# http://localhost:8137/index.html
```

## 기술
Three.js (WebGL), 저정밀 천문 계산(항성시·지평좌표 변환, 태양/달 위치), GLSL 셰이더.
